Instruction on how to use the app.





This application is designed to help people who would like to track their running exercise behaviour which includes the running route, running distance, running time and calorie burnt.

1. Before user starts the recording, user is prompt to enter some details regarding their body weight and type of running activity by clicking so that a more accurate data can be recorded.

2. Then, user can start the run by clicking the green start button.

3. User can press the red stop button if user would like to stop the recording temporary during the run.

4. After completing the run, user can click the blue finish button to finish the recording process and a simple summary window will pop up which shows user the details of the run.

5. User can then choose to save or delete the recording.

6. User can click view page link to view the history of saved records.
